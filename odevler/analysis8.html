Web Scrutiny and Evaluation Assignment

Example 1: Appealing Website

  • a. URL of the website: https://apple.com
  • b. Does this page use tables? No, this page does not use HTML tables for its layout. It relies on advanced layouts like Flexbox and CSS Grid.
  • c. Does this page use CSS? Yes, CSS is heavily used for responsive layouts, precise grid alignments, smooth text animations, and optimized media queries for mobile views.
  • d. Is this page appealing or unappealing? Highly Appealing.
    Reasons:
    1. It implements a beautiful minimalist design with balanced white space.
    2. The text typography is strong, dynamic, and exceptionally easy to read.
    3. The high-resolution product photography captures user engagement immediately.

Example 2: Unappealing Website

  • a. URL of the website: https://www.wikipedia.org
  • b. Does this page use tables? Yes, it uses HTML tables to structurally organize the large lists of languages, quick data links, and country portals.
  • c. Does this page use CSS? Yes, but it is only used for very basic text configurations, standard blue link colors, and primitive padding alignments.
  • d. Is this page appealing or unappealing? Unappealing.
    Reasons:
    1. The page layout is overcrowded with massive blocks of text links everywhere.
    2. The global visual identity feels outdated and lacks modern interface components (like custom cards or buttons).
    3. The absence of engaging background gradients or high-quality visuals makes it look plain and boring.
  • e. What would you do to improve it? To optimize the page layout, I would replace the old text links with a modern search-focused layout, group information using modern CSS card elements, add more negative padding space, and introduce an automated dark mode theme.
